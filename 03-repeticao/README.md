03 – Estruturas de Repetição
Este diretório contém exercícios práticos sobre estruturas de repetição em JavaScript.
O objetivo é entender como repetir blocos de código automaticamente usando diferentes tipos de laços.

---

Arquivos e conteúdos
for.js
Exemplo de repetição usando o laço for.

Utilizado quando sabemos quantas vezes o código deve se repetir.
O contador é controlado diretamente na estrutura do for.

Conceitos praticados:
Inicialização de contador
Condição de parada
Incremento automático
Execução repetida de código

---

while.js
Exemplo de repetição usando while.

O código é executado enquanto a condição for verdadeira.
A condição é verificada antes de cada repetição.

Conceitos praticados:
Controle manual de contador
Condição lógica
Possibilidade de loop infinito (se mal utilizado)

---

do-while.js
Exemplo de repetição usando do...while.

Diferente do while, esse laço executa o código pelo menos uma vez,
mesmo que a condição seja falsa na primeira verificação.

Conceitos praticados:
Execução garantida
Verificação da condição após a execução

---

controle-saldo.js
Exercício prático simulando um controle de saldo.

O programa reduz o saldo aos poucos até ele acabar,
mostrando mensagens no console a cada iteração.

Conceitos praticados:
Uso de laço de repetição em um cenário real
Atualização de variáveis
Condição de parada baseada em lógica de negócio

Exemplo de saída:
-Saldo atual: 80
-Saldo atual: 50
-Saldo atual: 40
-Saldo atual: Esgotado

---

## Objetivo do módulo

- Aprender a usar `for`, `while` e `do...while`
- Entender quando cada estrutura é mais adequada
- Desenvolver raciocínio lógico com repetição
- Criar base sólida para algoritmos mais complexos
