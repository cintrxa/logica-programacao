# 04 - Arrays

Neste módulo são abordados os fundamentos de **arrays em JavaScript**, estruturas usadas para armazenar e manipular listas de dados.

## Conteúdos praticados
- Criação e acesso a arrays
- Manipulação com `push` e `pop`
- Uso da propriedade `length`
- Percorrer arrays com estruturas de repetição
- Aplicação prática de lógica com controle de saldo

## Exemplos no diretório
- `arrays-basico.js`: conceitos básicos de arrays
- `controle-compras.js`: uso de arrays para simular compras e atualização de saldo

Esses conceitos são base para o desenvolvimento de APIs, manipulação de dados e lógica de back-end.
