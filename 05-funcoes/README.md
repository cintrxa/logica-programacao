# 05 - Funções

Neste módulo são apresentados os fundamentos de **funções em JavaScript**, essenciais para organização, reutilização de código e desenvolvimento de aplicações back-end.

## Conteúdos praticados
- Declaração de funções
- Uso de parâmetros
- Retorno de valores
- Aplicação de funções em fluxos de lógica
- Separação de responsabilidades

## Exemplos no diretório
- `funcoes-basico.js`: funções simples com e sem parâmetros
- `controle-funcoes.js`: uso de funções para controle de saldo e operações

Funções são a base para APIs, serviços e lógica de negócio.
