# 06 - Objetos

Este módulo apresenta o uso de **objetos em JavaScript**, fundamentais para representar entidades, estados e comportamentos em aplicações back-end.

## Conteúdos abordados
- Criação de objetos
- Propriedades e valores
- Alteração de estado
- Métodos e uso do `this`
- Aplicação prática em cenários reais

## Arquivos do módulo
- `objeto-basico.js`: estrutura básica de um objeto
- `alterar-objeto.js`: modificação de propriedades
- `metodos.js`: funções dentro de objetos
- `controle-usuario.js`: objeto aplicado a um fluxo real

Objetos são a base para models, serviços e lógica de negócio em APIs.
