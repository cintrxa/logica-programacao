# 07 - Arrays e Objetos

Neste módulo é abordada a combinação de **arrays e objetos**, estrutura central para manipulação de dados em aplicações back-end.

## Conteúdos abordados
- Listas de objetos
- Iteração e leitura de propriedades
- Aplicação de regras de negócio
- Estruturas semelhantes a dados de APIs

## Arquivos do módulo
- `lista-usuarios.js`: array contendo objetos
- `filtrar-ativos.js`: filtragem baseada em condição
- `controle-contas.js`: manipulação de dados financeiros
- `sistema.js`: objeto com lista interna e método

Este conteúdo prepara a base para uso de bancos de dados e APIs REST.
