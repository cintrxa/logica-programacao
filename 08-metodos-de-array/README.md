# 08 - Métodos de Array

Este módulo aborda os principais **métodos funcionais de array** em JavaScript, amplamente utilizados em aplicações back-end modernas.

## Conteúdos abordados
- `map`: transformação de dados
- `filter`: filtragem por condição
- `reduce`: acumulação e cálculos
- Aplicação em cenários reais

## Arquivos do módulo
- `map.js`
- `filter.js`
- `reduce.js`
- `controle-financeiro.js`

Esses métodos são essenciais para manipulação de dados vindos de APIs e bancos de dados.
