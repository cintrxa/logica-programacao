# 09 - Escopo e Funções

Este módulo aborda conceitos essenciais de **escopo de variáveis** e **funções modernas** em JavaScript, fundamentais para evitar bugs e escrever código previsível.

## Conteúdos abordados
- Diferença entre `var`, `let` e `const`
- Escopo de bloco e de função
- Funções tradicionais vs arrow functions
- Uso de funções com arrays

## Arquivos do módulo
- `escopo-variaveis.js`
- `funcoes.js`
- `escopo-funcao.js`
- `funcoes-e-arrays.js`

Dominar esses conceitos é essencial para desenvolvimento back-end com Node.js.
