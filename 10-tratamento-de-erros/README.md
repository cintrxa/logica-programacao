# 10 - Tratamento de Erros

Este módulo apresenta os principais conceitos de **tratamento de erros em JavaScript**, fundamentais para aplicações seguras e estáveis.

## Conteúdos abordados
- Uso de `try` e `catch`
- Lançamento de erros com `throw`
- Validação de dados
- Simulação de cenários de API

## Arquivos do módulo
- `try-catch.js`
- `throw-error.js`
- `validacao.js`
- `api-simulada.js`

O tratamento de erros é essencial para garantir confiabilidade em sistemas back-end.
